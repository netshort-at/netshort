# netshort.at

**Österreichische Netto-Short-Positionen (FMA-Daten)**

## Ziel
Transparente Darstellung von Short-Positionen österreichischer Unternehmen. Historische Zeitreihen und institutionelle Anteile.

## Nutzung
- Filter nach Aktie & Institution
- Gestapelte Balken-Charts mit historischen Höchst- und Tiefpunkten
- Schwellenwert <0,2% wird herausgefiltert
- Hinweis: Meldeschwellen & Cancellation bedeuten, dass nicht alle Positionen sichtbar oder endgültig beendet sein müssen.

## Rechtliches
- Keine Anlageberatung
- Privates, nicht-kommerzielles Projekt
- Haftung ausgeschlossen
- KI-Unterstützung: Projekt mithilfe von ChatGPT erstellt

## Datenquelle
[Finanzmarktaufsicht (FMA)](https://www.fma.gv.at/)
